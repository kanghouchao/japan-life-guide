# Japan Life Guide

このプロジェクトは、日本に長期滞在するつもりの外国人のために、日本の基本的なルールや手続きについての情報を提供するウェブサイトを作成することを目的としています。具体的には、以下の内容を含みます：

- 登録手続き
- 電話カードの購入方法
- 銀行カードの取得方法
- ゴミの分別と処理方法
